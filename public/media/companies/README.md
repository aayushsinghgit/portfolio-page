# Company Logos

Place company logo images in this folder:

- infosys.png - Infosys Limited logo
- appen.png - Appen logo  
- soulhq.png - Soulhq.io logo
- turing.png - Turing logo
- braintrust.png - Braintrust logo

**Recommended specifications:**
- Format: PNG with transparent background
- Size: 200x200px or similar square dimensions
- Color: White/light colored logos work best on dark background
